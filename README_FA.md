# نسخه اندرویدی نرم افزار رقم

این پروژه یک اپ اندرویدی WebView است که به نسخه سروری نرم افزار رقم متصل می‌شود. اطلاعات اصلی در سرور و SQLite نگهداری می‌شوند و اپ اندروید نقش رابط موبایلی را دارد.

## امکانات

- درخواست آدرس سرور در اولین اجرا
- امکان تغییر آدرس سرور از منوی بالای برنامه
- ورود مدیر و کاربران از همان صفحه ورود نرم افزار رقم
- پشتیبانی از انتخاب فایل JSON برای بازیابی مدیر
- ذخیره فایل پشتیبان JSON در پوشه `Downloads/Ragham`
- پشتیبانی از چاپ گزارش و ذخیره PDF با سرویس چاپ اندروید
- پشتیبانی از دکمه بازگشت اندروید
- امکان تازه‌سازی، خروج از حساب و بازکردن سامانه در مرورگر
- اتصال به HTTP برای تست شبکه داخلی و HTTPS برای سرور عمومی

## استفاده روی گوشی

1. ابتدا نسخه سروری رقم را اجرا کنید.
2. گوشی باید بتواند به سرور دسترسی داشته باشد.
3. اپ را نصب و اجرا کنید.
4. در اولین اجرا آدرس را وارد کنید، مانند:

```text
http://192.168.1.20:8080
```

برای سرور اینترنتی بهتر است آدرس HTTPS وارد شود:

```text
https://ragham.example.com
```

## ساخت APK با Android Studio

1. Android Studio را نصب کنید.
2. گزینه **Open** را بزنید و پوشه این پروژه را انتخاب کنید.
3. منتظر بمانید Gradle Sync کامل شود.
4. از منوی **Build > Build Bundle(s) / APK(s) > Build APK(s)** استفاده کنید.
5. فایل APK در مسیر زیر ساخته می‌شود:

```text
app/build/outputs/apk/debug/app-debug.apk
```

## ساخت APK با GitHub

این پروژه دارای فایل GitHub Actions است. پس از بارگذاری پروژه در GitHub:

1. وارد زبانه **Actions** شوید.
2. گردش‌کار **Build Android APK** را باز کنید.
3. گزینه **Run workflow** را بزنید.
4. پس از پایان ساخت، فایل `ragham-android-debug-apk` را از بخش Artifacts دانلود کنید.

## نکته امنیتی

برای استفاده عمومی از HTTPS استفاده کنید. مجوز HTTP فقط برای آزمایش روی شبکه داخلی فعال شده است. احراز هویت واقعی و کنترل دسترسی باید در نسخه سروری نیز فعال باشد.
